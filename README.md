# React Typescript Starter

## Features

- use browser cache for react, don't cache react component
- React + Typescript, It's simple

## Install

```
yarn install
```

## Usage

serve app and reload automatically
```
yarn start
```

build from src
```
yarn build
```

## Directory Structure

```
.
```
